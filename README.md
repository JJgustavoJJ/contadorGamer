# 🎮 contadorGamer 

Projeto desenvolvido em React Native com Expo.

---

# Preview do APP

o aplicativo possui:
* contador de pontos;
* botão de aumentar;
* botão de diminuir;
* botão de resetar;
* interface estilo gamer.

---

# Tecnologias utilizadas

* React Native
* Expo
* JavaScript



# Como executar o projeto

## 1. clonar o repositorio
```
git clone https://github.com/JJgustavoJJ/contadorGamer.git
```


## 2. Entrar na pasta de projeto
```
cd contador
```


## 3. Instalar dependência
```
npm install
```


## 4. Rodar o projeto
```
npm start
```
